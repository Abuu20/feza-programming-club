# Feza Programming Club Platform

A complete web application for managing a school programming club with activities, challenges, Python editor, and more.

## Features
- ✅ Student registration with admin approval
- ✅ Admin dashboard for content management
- ✅ Activities with image upload and registration
- ✅ Coding challenges with leaderboard
- ✅ Python practice with code editor
- ✅ Gallery with image upload
- ✅ Announcements system
- ✅ Contact form
- ✅ Responsive design for all devices

## Tech Stack
- React
- Tailwind CSS
- Supabase (Database, Auth, Storage)
- Monaco/CodeMirror Editor
